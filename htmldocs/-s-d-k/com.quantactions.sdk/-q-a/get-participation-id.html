getParticipationId

fun getParticipationId(context: Context): Flow<QAResponse<PartIdResponse>>

Returns an object of type PartIdResponse that contains the participation ID of the study to which the device is currently subscribed to, if multiple devices are subscribed using the same participationId it returns all the device IDs.
