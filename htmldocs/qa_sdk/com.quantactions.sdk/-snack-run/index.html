com.quantactions.sdk.QARunnable

This QARunnable example shows how to add an alert dialog to the runnable, in this way the user can be notified that the app is performing a task in the background, when done the dialog is dismissed and a message appears in the SnackBar. You can create the Runnable with custom messages for success and failure. Created by Enea Ceolini on 23/08/17. Contact: enea.ceolini@quantactions.com
