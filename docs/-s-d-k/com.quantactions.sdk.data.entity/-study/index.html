Study

@JsonClass(generateAdapter = true)
data class Study(val studyId: String, val privacyPolicy: String?, val studyTitle: String?, val dataPattern: String?, val gpsResolution: Int, val canWithdraw: Int, val syncOnScreenOff: Int?, val perimeterCheck: Int?, val permAppId: Int?, val permDrawOver: Int?, val permLocation: Int?, val permContact: Int?)

When subscribing to a study with a participation ID or a study ID, the call will return an object of this class which contains all information related to the study including necessary permissions and privacy policy.
