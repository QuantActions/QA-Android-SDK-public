getSleepSummarySample

fun <T> getSleepSummarySample(context: Context, apiKey: String, score: Metric<T>, from: Long = 0, to: Long = Double.POSITIVE_INFINITY.roundToLong()): Flow<TimeSeries<T>>

Get a SleepSummary relative to a fictitious test device. The function returns an object of type TimeSeries.SleepSummaryTimeTimeSeries which contains timestamps and values of the sleep summary. The call is asynchronous ans returns a flow. You can use this function to test your data workflow and visualization.
