getJournalEvents

fun getJournalEvents(): List<JournalEvent>

Retrieves the Journal events, meaning the events that one can log together with a journal entry. The events come from a fixed set which may be updated in the future, this function return the latest update to the JournalEvent.

Return

A flow with a list of JournalEvent

Throws

SDKNotInitialisedException

if the SDk was not initialised before.
